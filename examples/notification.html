Notification 通知提醒 1.2.2及以上版本支持

在页面的右上角显示通知提醒信息，常用于以下场景：较复杂的通知内容、有交互的通知以及系统的推送，请注意这组件是1.2.2版本以后添加，如果使用请升级版本到1.2.2 版本

基础样式 #

基本使用方法，两种类型的信息提示，默认4秒后自动关闭，调用方法：notification.alert(options)，具体配置信息可见下面options配置项
<code>

Nv.use("notification",function(nv,notification){ document.getElementById("notification_all_1").addEventListener("click",function(){ notification.alert({ title:"Title", content:"This is the content......" }) },false) document.getElementById("notification_title_1").addEventListener("click",function(){ notification.alert({ title:"Only title" }) },false) });
显示代码

带有图标的通知提醒 #

图标可以让通知提醒更醒目，分别有信息提醒notification.info(options)，成功notification.success(options)，警告notification.warning(options)，错误提示notification.error(options)
<code>

Nv.use("notification",function(nv,notification){ document.getElementById("notification_info_1").addEventListener("click",function(){ notification.info({ title:"Info title", content:"This is the info content......" }) },false) document.getElementById("notification_success_1").addEventListener("click",function(){ notification.success({ title:"Success title", content:"This is the success content......" }) },false) document.getElementById("notification_warning_1").addEventListener("click",function(){ notification.warning({ title:"Info warning", content:"This is the warning content......" }) },false) document.getElementById("notification_error_1").addEventListener("click",function(){ notification.error({ title:"Error title", content:"This is the error content......" }) },false) });
显示代码

自定义显示时长 #

默认时长为4秒，为 0 则不自动关闭。配置信息详情查看options配置信息
<code>

Nv.use("notification",function(nv,notification){ document.getElementById("notification_error_1").addEventListener("click",function(){ notification.success({ title:"Self time title", content:"Forever show ~~~ by options.time", iconsClassName:"nvicon-transaction", time:0 }) },false) });
显示代码

options配置 #

属性
类型（是否必要）
描述
options.time
Number (选填)
定义提醒框消失时间，默认是4000毫秒
options.title
String (必填)
显示提示信息的标题，如果没有标题可以替换为内容，内容区content为空即可
options.content
String (选填)
提示内容
options.iconsClassName
String (选填)
提示图标，传入ICONS中的任何样式即可。e.g:"nvicon-star"
options.iconsShowStatus
Number/Boolean
是否显示图标，可以是数字0、1,也可以是布尔值
