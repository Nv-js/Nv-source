Message 全局提示

用户主动操作后的反馈提示

基础样式 #

基本使用方法，四种类型的信息提示，默认3s消失，可自定义时长，不要超过15个字
<code>

Zr.use("message","jquery",function(zr,message,$){ $(".info").off("click").on("click",function(){ message.info("This is a message of info"); }) $(".success").off("click").on("click",function(){ message.success("This is a message of success"); }) $(".warning").off("click").on("click",function(){ message.warning("This is a message of warning"); }) $(".error").off("click").on("click",function(){ message.error("This is a message of error"); }) })
显示代码

自定义显示时长 #

含有辅助性文字介绍的警告提示
<code>

Zr.use("message","jquery",function(zr,message,$){ $(".info").off("click").on("click",function(){ //10s后消失 message.info("This message will disappear in 10 seconds",{ time:10000 }); }) })
显示代码

加载中 #

全局Loading的状态，并异步在某个时机移除。
<code>

Zr.use("message","jquery",function(zr,message,$){ $(".info").off("click").on("click",function(){ //10s后消失 message.loadingShow("Loading..."); //message.loadingHide() }) })
显示代码

警告提示框 #

经常用于阻断性消息提醒，可以自定义图标
<code>

Zr.use("message",function(zr,message){ message.alert('@注意此时是对话框') })
显示代码

确认提示框 #

确认是否要进行下一步操作，常用语警醒确认操作
<code>

Zr.use("message",function(zr,message){ message.confirm('是否要删除此条数据') })
显示代码
