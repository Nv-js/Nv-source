Drop down 下拉菜单

点击或移入操作区，会向下弹出一个菜单列表。

使用方法

  • 当页面上的操作命令过多时，用此组件可以收纳操作元素。
  • 点击或移入触点，会出现一个下拉菜单。

基础样式 #

最简单的下拉样式
<code>
Dropdown
  • Menu 1（normal）
  • Menu 2（normal）
  • Menu 3（disable）
  • Menu 4（normal）
Dropdown
  • Item 1
  • Item 2
  • Item 3（disable）
  • Item 4（normal）

触发方式 #

最基本的触发方式是 hover ，可以更改为 click 触发。添加zr-event属性有"click"、"hover"(默认)

  • Item 3（disable）
  • Item 4（normal）

  • Item 3（disable）

  • Item 4（normal）

多级菜单 #

菜单多用于功能较多的时候使用，建议最多二级。添加的子元素菜单只需要添加样式为"zr-dropdown-children"即可。
<code>
Cascading menu
  • List 1
    • Child 1
    • Child 2
  • List 2
    • Child 1 target blank
    • Child 2
    • Child 3（disable）
    • Child 4
Cascading menu
  • List 1
    • Child 1
    • Child 2
  • List 2
    • Child 1 target blank
    • Child 2
