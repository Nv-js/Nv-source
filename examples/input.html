Input 输入框

通过鼠标或键盘输入内容，可组合使用，常用于查询条件、表单、新建或修改等场景。

使用方法

  • 文本框一般用于接收较短的字符串，如账号、密码、姓名等。文本框高度和宽度固定，显示的文本一般左对齐。
  • 当用户输入的内容超出文本框范围时显示最后输入的文本位置，横向滑动可查看已输入的内容。如要显示较长的字符串，请改用“多行文本框”。
