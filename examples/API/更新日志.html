严格遵循 Semantic Versioning 2.0.0 语义化版本规范

发布周期 #

版本格式：主版本号.次版本号.修订号，先行版本号及版本编译信息可以加到“主版本号.次版本号.修订号”的后面，作为延伸。
  • 主版本号：当你做了不兼容的 API 修改
  • 次版本号：当你做了向下兼容的功能性新增
  • 修订号：当你做了向下兼容的问题修正
