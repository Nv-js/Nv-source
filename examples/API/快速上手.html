第二步，页面适当位置引入，建议放到body节点之前，引入方式：直接引入"//storage.jd.com/1.3.3/zr.min.js"的脚本文件

"//storage.360buyimg.com/1.3.3/zr/css/cdn_zr.min.css"
"//storage.360buyimg.com/1.3.3/zr.min.js"
CDN引入建议使用HTTPS方式，防止某些区域HTTP被挟持
第三部，配置信息（建议放到第一步后面即可）：
Javascript 代码
<script type="text/javascript"> //配置方法 Zr.config({ //配置plugin_modules的目录； baseUrl:"/js/plugin_modules/", //开发模式，默认关闭； requestTime:true }) //初始入口，Zr各个模块准备就绪后调用； Zr.ready(function(){ //使用相应的模块； Zr.use("jquery","./js/code",function(zr,$,code){ //自动注入了jquery组件； //自动注入了./js/code组件； }) }) </script>

简要说明 #

总体来说，Zr采用的是异步加载策略，当需要使用的时候才加载。下面来详细讲解常用方法 Zr.use 和 Zr.add 以及其他工具类方法
Method

Zr.use #

语法：Zr.use(param1,param2,[...],callback)
属性
类型（是否必要）
描述
param1,param2,...
String（必填）
入参参数，用逗号间隔，可以使用配置的别名与Zr.add(name,...)中的name名称即可
callback
Function（选填）
回调函数，e.g : [function(zr,callback1,callback2,[...])]
1.回调函数第一参数为zr组件生态环境引用，可以直接访问属性并修改，请注意修改时候注意不要引起变量泛滥。
2.callback1,callback2,[...]分别对应param1,param2,[...]返回的内容
Javascript:
Zr.use("jquery","page","./timer/theme.js",function(zr,$,page,theme) //todo something； $(function(){ page.init({ total:500 }); theme.change(); }) })

Zr.add #

语法：Zr.add(name,callback,options)
属性
类型（是否必要）
描述
name
String（必填）
此name属性为相对于目录"plugin_modules"文件下的位置
常用使用模块方式名称："./code/index"或者'./code/index.js'
callback
Function（选填）
回调函数，第一个参数为zr的全局引用，之后的参数分别对应引用配置
options
Object（选填）
回调函数，e.g : [function(zr,callback1,callback2,[...])]
1.回调函数第一参数为zr组件生态环境引用，可以直接访问属性并修改，请注意修改时候注意不要引起变量泛滥。
2.callback1,callback2,[...]分别对应param1,param2,[...]返回的内容
3.配置对象，其中唯一参数为"requires",数组类型，此处相对的目录位置是当前文件的位置为起点
Javascript:
//根据依赖，第一个参数永远为zr，之后都是根据依赖顺序注入； Zr.add("./mobile/",function(zr,jquery,theme){ //todo something； },{ //属性配置，依赖哪些模块； requires:["jquery","/map/theme"] })

Zr.config #

语法：Zr.config(opt)，初始化配置方法,opt为初始化对象配置信息，为Object类型
属性
类型（是否必要）
描述
opt.baseUrl
String（必填）
配置"plugin_modules"的根路径位置，这个常用于开发的代码库地址
opt.requestTime
Boolean（选填）
生成随即时间戳，防止缓存，开发环境建议打开，上线后必须关闭
opt.version
String（选填）
当前Zr的版本信息,目前最新版本0.0.1
opt.language
String（选填）
当前语言环境，设置将影响组件的显示
opt.requestVersion
String（选填）
用于更新线上环境所有Zr的异步组件当前版本信息，常用于发版清除缓存
opt.cdnUrl
String（选填）
指向公共资源库地址，文件命名格式："cdn_**.js"即可，默认指向：//storage.360buyimg.com/0.0.1/
opt.module
Object{name1:value1,name2:value2,......}（选填）
Zr.use使用时候对应的别名：name对应别名名称，value对应的别名相关配置
Zr.config({
"module":{
//模块别名名称
"jquery":{
//配置的寻址路径
"path":"./jquery/"
},
"jquery3":{
"path":"./jquery/jquery3.js"
},
"zepto":{
"path":"./zepto/"
}
}
})
Javascript:
Zr.config({ //根目录配置路径; "baseUrl":"/plugin_modules/", //文件请求是否添加后缀; "requestTime":false, //模块别名配置区域; "module":{ //模块别名名称; "jquery":{ //配置的寻址路径; "path":"./jquery/" }, "jquery3":{ "path":"./jquery/jquery3.js" }, "zepto":{ "path":"./zepto/" } } })

Zr.extend #

语法：Zr.extend(boolean,object1,object2)，copy合并继承，不会影响原来对象，后者与前者某一个属性重复，后者则覆盖前者对象
属性
类型（是否必要）
描述
boolean
Boolean（选填）
是否要深度copy，默认是false
object1
Object（必填）
需要合并的对象
object2
Object（必填）
需要合并的对象
Javascript:
var defaults = { a:1, b:{ status:false, show:false, value:100 } } var options = { a:100, b:{ status:true, name:"现状" }, c:2 } var opt = Zr.extend(defaults,options); console.log(opt); /** * 输出结果为 * { * a:100, * b:{ * status:true, * name:"现状" * }, * c:2 * } * * */ opt = Zr.extend(true,defaults,options); /** * 输出结果为 * { * a:100, * b:{ * status:true, * show:false, * name:"现状" * value:100, * }, * c:2 * } * * */

Zr.tools #

语法：Zr.tools，对象Object类型，工具类，方便实用的小工具
方法名称
描述
bind(func,object,[param1,param2,...])
对象绑定，借用构造函数
/**
* {param} (func) [function] 被绑定的函数对象
* {param} (object) [object] 绑定指向的对象
* {param} (param1,param2,...) [入参] 绑定后参数一函数（func）执行后的形式参数
* e.g:
* var func = function(greeting){ return greeting + ': ' + this.name };
* func = tools.bind(func, {name: 'moe'}, 'hi');
* func();
* => 'hi: moe'
* */
each(object,callback)
循环对象或者数组
/**
*循环执行object对象（数组或者对象都可以），参数二回调方法中返回两个参数
*其中回调参数一是数组的下角标或者对象的key
*参数二为数组小角标对应的数值本身或者对象的value
*/
endWidth(str,suffix)
判断参数一是否以参数二结尾。主要是针对文件路径的判断
extname(boolean,object1,object2)
获取当前入参的路径别名
isArray(Object)
是否是数组
isBoolean(Object)
是否是布尔值
isDate(Object)
是否是日期类型
isEmptyObject(Object)
是否是空对象
isFunction(Object)
是否是方法
isNull(Object)
是否是空对象
isNumber(Object)
是否是数字
isObject(Object)
是否是对象
isPlainObject(Object)
是否是Zr的对象
isRegExp(Object)
是否是正则
isString(Object)
是否是字符类
isUndefined(Object)
是否是空对象
key(Object)
获取对象的key值返回数组
later(callback,time,boolean)
定时器，支持单次、多次、取消
/**
* {param} (callback)
* 回调函数内部执行方法
* {param} (time) [number]
* 间隔执行时间
* {param} (boolean) [boolean]
* 默认是false执行一次，true后触发循环执行
* {return} (callback) [object]
* 返回对象中包含以下方法
* (id) [number] 当前定时器的唯一标识
* (interval) [boolean] 当前是单次执行（false）还是循环执行（true）
* (cancel) [function] 取消当前定时器方法
* */
makeArray(object)
伪数组转换成数组
size(object)
数组返回长度，对象返回关键字key的长度
stringArray(object)
字符串转转换成数组
trim(stringObject)
字符串去首尾空格
values(object)
获取对象的value部分，返回数组
cookie(key,value,time)
/*获取cookie与设置cookie
*{param} (key)
*cookie的key，必填
*{param} (value)
*设置cookie时候使用，如果没有此参数，则是获取cookie
*{param} (time)
* 设置cookie过期时间默认是30,可以是0.1 或者 1、2
*/
storage(key,value,which)
/*获取离线存储与设置离线存储数据
*{param} (key)
*离线存储的key，必填
*{param} (value)
*设置离线存储时候使用，如果没有此参数，则是获取数据
*{param} (which)
* 设置离线存储的类型true为sessionStorage / false（默认）为localStorage
*/
browser
浏览器特征检测，非常实用
/*代理检测的一种手段
------------------
*browser.versions
**返回对象，检测对象包
**browser.versions.trident //IE内核 是则返回true
**browser.versions.presto //opera内核 是则返回true
**browser.versions.webKit //苹果、谷歌内核 是则返回true

**browser.versions.android //android终端或者uc浏览器 是则返回true

**返回当前环境默认语言
*/
format(fmt,time)
日期格式化
//fmt表示格式类型："yyyy-MM-dd hh:MM:ss SSSS 季度:q"
