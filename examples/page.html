Pagination 分页

当数据量较多一屏展示不下时，可使用分页快速进行数据切换。

使用方法

  • 用户知悉页面总量，支持在当页或其他页面进行进一步操作。

基础样式 #

最基本的分页样式，当分页小于5页时，全部展示(通过 items_per_page 参数控制每页显示条数)。
<code>
Zr.ready(function(){ Zr.use("./zrModules/page",function(zr,page){ page.init(200,{ items_per_page:40 },document.getElementsByClassName("basis-page")[0]); }) })
显示代码

更多分页 #

当分页大于5页时，部分页码隐藏(参数 items_per_page 默认为10条/页)
<code>
Zr.use("./zrModules/page",function(zr,page){ page.init(200,{ items_per_page:10 },document.getElementsByClassName("more-page")[0]); })
显示代码

可选择每页数量 #

可以选择每页显示列表数量(通过 select_page 参数进行控制，默认为false不显示)
<code>
Zr.use("./zrModules/page",function(zr,page){ page.init(200,{ select_page:true },document.getElementsByClassName("select-page")[0]); })
显示代码

可跳转至某页 #

出现跳转输入框,输入页数按下enter键即可跳转到相对应的页数
<code>
Zr.use("./zrModules/page",function(zr,page){ page.init(200,{ select_page:true, jump:true },document.getElementsByClassName("jump-page")[0]); })
显示代码

显示总数 #

可选择是否显示总数(通过 total 参数进行配置，默认为false不显示)
<code>
Zr.use("./zrModules/page",function(zr,page){ page.init(200,{ select_page:true, jump:true, total:true },document.getElementsByClassName("total-page")[0]); })
显示代码

MINI页码 #

去除了翻页按钮边框的简洁型翻页。
<code>
Zr.use("./zrModules/page",function(zr,quickMenu,code,page){ page.init(200,{ select_page:true, jump:true, total:true, simple:true },document.getElementsByClassName("simple-page")[0]); })
显示代码

简版翻页 #

只显示跳转输入框和上下页按钮的简单翻页。
<code>
Zr.use("./zrModules/page",function(zr,page){ page.init(200,{ mini:true },document.getElementsByClassName("mini-page")[0]); })
显示代码

配置 #

page组件可以通过传递不同的参数来配置你所需要的翻页功能(不传递参数则显示默认page样式)

参数配置 #

page.init(total_number,{},dom)
属性
类型（是否必要）
描述
items_per_page
Number（选填）
每页显示多少条数据，默认是10条
num_display_entries
Number（选填）
显示多少个翻页按钮，默认是6个
current_page
Number（选填）
初始化插件从第几页开始，默认从第一页开始
link_to
String（选填）
翻页按钮的a表现指向，默认为"javascript:;"
first_text
String（选填）
首页按钮，文字可自定义，当为空时字符串不显示首页
last_text
String（选填）
尾页按钮，文字可自定义，当为空时字符串不显示首页
prev_text
String（选填）
尾页按钮，文字可自定义，初始默认为<
next_text
String（选填）
尾页按钮，文字可自定义，初始默认为>
ellipse_text.
String（选填）
分页中间的分割按钮，初始默认是...
prev_show_always
Boolean（选填）
是否显示上一页,初始默认显示
next_show_always
Boolean（选填）
是否显示上一页,初始默认显示
page_callback
Function（选填）
触发翻页按钮的回调函数，返回页数和翻页指向的dom节点
selected_page_callback
Function（选填）
触发页数下拉框的回调函数，返回每页多少条数据和当前dom节点
select_page
Boolean（选填）
是否出现页数下拉框，默认出现
jump
Boolean（选填）
是否出现跳转，默认出现
total
Boolean（选填）
是否显示总页数，默认出现
complete
Boolean（选填）
是否出现完整版分页，默认是true
simple
Boolean（选填）
是否出现简单版分页，默认是false
mini
Boolean（选填）
是否出现mini版分页，默认是false
