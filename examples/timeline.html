Time 时间轴

按时间顺序垂直展示的信息 API文档 组件问题反馈和讨论

基础样式 #

最基本的时间轴，只需结构保证为外层是 nv-timeline 样式，每个条目是样式 nv-timeline-item 即可
<code>

  • 2018-01-11

    Approve and initiate a project

    project description

  • 2018-01-03

    Require review

  • 2018-01-01

    UE&UI design

  • 2018-01-11

    Approve and initiate a project

    project description

  • 2018-01-03

    Require review

  • 2018-01-01

    UE&UI design
显示代码

状态提示 #

绿色样式 nv-timeline-green 用于已完成、成功状态、红色样式 nv-timeline-error 表示错误、橙色样式 nv-timeline-warning 表示警告状态、 蓝色为默认色可表示正在进行
<code>

  • 2018-01-21

    Approve and initiate a project

    project description

  • 2018-01-13

    Require review

  • 2018-01-11

    UE&UI design

  • 2018-01-01

    Front-end development

  • 2018-01-21

    Approve and initiate a project

    project description

  • 2018-01-13

    Require review

  • 2018-01-11

    UE&UI design

  • 2018-01-01

    Front-end development
